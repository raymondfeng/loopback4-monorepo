# loopback4-monorepo

This is the [lerna](https://github.com/lerna/lerna) monorepo to manage multiple

## Getting started

### Check out the git repository

### Install dependencies and bootstrap the monorepo

```sh
npm ci
```

### Build docker images

```sh
npm run docker:build
```
